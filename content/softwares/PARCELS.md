# OceanParcels : A highly customisable Lagrangian simulation framework


## About OceanParcels  *(adapted from the parcels website)*

Parcels provides a set of Python classes and methods to create customisable particle tracking simulations using gridded output from (ocean) circulation models.

+ Documentation, related bibliography, sources and installation instructions are available on the [parcels website](https://parcels-code.org/).
